Goal

Connect mlr3 to Hugging Face datasets and convert a HF dataset to an mlr3::Task.

Summary of Approach

Option A (Prototype):

Use a simple CSV download approach for HF repos with CSV files. This is implemented in the prototype function htsk() which accepts a shorthand hf:owner/repo/path/to/file.csv and downloads the resolved file from https://huggingface.co/datasets/<owner/repo>/resolve/main/<path>.

Option B (Recommended for Full Support):

Use reticulate to call Python's datasets or the hfhub client to support multiple dataset formats and streaming. This avoids loading whole datasets into R memory.

Implementation (Prototype htsk())

library(mlr3)
library(mlr3hf) # your package

# Example: simple hf: shorthand
task <- htsk('hf:username/repo/path/to/file.csv', target = 'label')
print(task)

Justification & Methods

Design choices include:

  • Memory-safety via streaming: Avoid loading entire datasets into memory
  • Use of reticulate: Robust Hugging Face access
  • Fallback prototype: CSV-hosted datasets support
